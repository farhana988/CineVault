## ğŸ§ª Development & Running Locally

### Clone the Repository

```bash

```

### Install Dependencies

```bash
npm install
```

### Create a .env.local file and add:

```bash
NEXT_PUBLIC_TMDB_API_KEY=your_tmdb_api_key
NEXT_PUBLIC_TMDB_BASE_URL=https://api.themoviedb.org/3
```

### Run the Frontend

```bash
npm run dev
```
